Você pode colocar esse site no GitHub Pages, que é grátis e fácil. Passo a passo:

  1. Criar conta no GitHub.

  2. Clique em New Repository (novo repositório).

    • Nomeie, por exemplo: eco-denuncia.

    • Deixe como público e crie.

  3. No seu computador, crie uma pasta e coloque dentro o arquivo index.html (que eu te dei).

  4. Entre nessa pasta e clique com botão direito → Git Bash Here (ou use terminal).

    • Digite:

      bash
      git init
git add .
git commit -m "primeira versão"
git branch -M main
git remote add origin https://github.com/SEU_USUARIO/eco-denuncia.git
git push -u origin main

  5. No GitHub, vá em Settings → Pages.

    • Em “Source”, escolha main branch e salve.

  6. Ele vai gerar um link do tipo:

    arduino
    https://seuusuario.github.io/eco-denuncia/

Esse link já é seu site online. 🚀
